# ⚡ Radar de Recul Intelligent (Arduino)

Bienvenue dans mon laboratoire virtuel ! Ce projet présente un système de sécurité complet et autonome combinant détection globale et télémétrie de précision.

📺 **[Voir le site vitrine interactif en direct](https://ton-pseudo.github.io/radar-recul-arduino/)**

## 🚀 Comment l'utiliser chez vous ?

1. **Télécharger le code :** Cliquez sur le bouton vert **Code** en haut de cette page, puis sur **Download ZIP**.
2. **Le circuit :** Ouvrez le fichier `circuit.png` pour voir le schéma complet des connexions ou cliquez sur le lien Tinkercad présent sur le site web pour lancer la simulation.
3. **Le Firmware :** Copiez le code présent dans le fichier `index.html` (section Code source) et téléversez-le dans votre carte Arduino Uno via l'IDE Arduino officielle.

## 🛠️ Composants requis
* 1x Arduino Uno R3
* 1x Capteur Ultrasonique HC-SR04
* 1x Capteur de mouvement PIR
* 1x Écran LCD 1602A
* 1x Potentiomètre B10K
